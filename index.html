We are a group of developers on GitHub, we network on Discord. We have a few repos you can contribute to and a couple of Project Boards you can view and work on once you are a member.

Currently we are working on the "hasidicdevs.github.io" repository. It is done in Bootstrap, but we are making the transition over to React.js We also have a discord bot base project. You can download the released version of the code and all you need to do is create your .env file and npm install then you are able to work on it and expand it as you need. We have web developers, software developers, graphic designers and more. Come participate in the server and get to know us.

As we grow bigger and gain members, we will have a better idea of what this group is all about. For now we just like to code and chat together. If you have a project you need help on feel free to tag us on GitHub @hasidicdevs/web-development and we can jump in and help. Same goes for @hasidicdevs/graphics if you need logo or graphics help.
